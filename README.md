# Описание<br>
## Это веб-приложение на Django, разработанное для обучения английскому языку через интерактивные викторины. Приложение предоставляет структурированный учебный процесс с мгновенной обратной связью, делая изучение языка увлекательным и эффективным. Оно решает распространённую проблему запоминания лексики, используя принципы интервального повторения и геймификацию.
<h1> Основные функции </h1>
<p> Интерактивные викторины: Вопросы с выбором ответа и мгновенной обратной связью <br>
Отслеживание результатов: Реальное время подсчёта баллов и итоговых результатов <br>
Тематическое обучение: Организовано по темам <br>
Удобный интерфейс: Чистый и интуитивно понятный дизайн с адаптивной вёрсткой <br>
Хранение данных: Результаты викторин сохраняются в SQLite базе данных <br>
Источник данных в формате YAML: Вопросы и ответы управляются через структурированные YAML-файлы </p>
 Структура проекта <br>

```text
quiz/
├── core/               # Основная конфигурация Django
├── sim/                # Основное приложение
│   ├── migrations/     # Миграции базы данных
│   ├── templates/      # HTML-шаблоны
│   │   └── partials/   # Шаблоны викторины (начало, вопрос, ответ, завершение)
│   ├── admin.py
│   ├── apps.py
│   ├── models.py
│   ├── tests.py
│   ├── urls.py
│   └── views.py
├── quiz_data.yaml      # Источник вопросов и ответов
├── db.sqlite3          # База данных SQLite
├── manage.py
└── .gitignore
```
<h2> Установка и настройка </h2>
Предварительные требования <br>
Python 3.10+ <br>
pip </p>
Шаги установки<br>
Клонируйте репозиторий:

```text
git clone https://github.com/ваш-логин/english-quiz.git
cd english-quiz
```
Создайте и активируйте виртуальное окружение:

```text
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```
Установите зависимости:
```text
pip install -r requirements.txt
```
Примените миграции базы данных:

```text
python manage.py migrate
```
Загрузите начальные данные викторины:

```text
python manage.py loaddata quiz_data.yaml
```

Запустите сервер разработки:

```text
python manage.py runserver
```

Откройте приложение в браузере:

```text
http://127.0.0.1:8000/
```

<h2> Как пользоваться </h2>
Начать викторину: Выберите "Английский язык (10 вопросов)" и нажмите "Начать" <br>
Отвечайте на вопросы: Выберите ответ и нажмите "Ответить" <br>
Просмотр результатов: Сразу видите обратную связь с указанием правильных/неправильных ответов <br>
Завершение викторины: Посмотрите итоговый балл и процент правильных ответов после всех вопросов <br>
Новый запуск: Нажмите "Начать новый", чтобы пройти викторину заново <br>
Технические особенности
Шаблоны Django: Использование наследования шаблонов с блоками для единообразного оформления <br>
Валидация форм: Надёжная проверка данных с понятными сообщениями об ошибках <br>
Управление данными через YAML: Контент викторины хранится в структурированном YAML для удобного редактирования <br>
Адаптивный дизайн: Чистый интерфейс с использованием Bootstrap для мобильной версии <br>
Подсчёт баллов: Реальное время расчёта результатов с процентным соотношением <br>
